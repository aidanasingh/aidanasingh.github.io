# Run Length Encoder

[This program](https://github.com/aidanasingh/operating-systems-exercises/blob/main/run_length_encoder/multithreaded_encoder.c) is a parallelized command line tool that compresses text files by encoding repeated characters as a character and its run length (e.g. aaaaaa -> a6). It takes arguments for any number of files to be encoded into one output file, and one argument for the number of concurrent threads following -j.
